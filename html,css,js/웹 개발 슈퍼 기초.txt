<2. HTML이란 ? sublime text 설치>

<3. HTML 조금 더 연습해보기>
html은 골격
head는 헤드부분, body는 바디부분, html로 감싸인 부분이 전체 html이라 생각하면 됨
title은 브라우저의 텝의 이름

h1 ~ h6 등으로 글씨 크기 조절, 1이 제일 큰거고 숫자 커질수록 작아짐

a, href  이용해서 링크 걸기

li는 리스트를 만드는거고 ul은 리스트를 묶어줌




<4. HTML 조금 더 more 연습해보기>
img, src 이용해서 사진 넣기
	사진 검색 후 이미지 주소 복사해서 src에 넣어줌, src 뒤에 width와 heigth로 크기 조절
input으로 해서 빈칸 넣을 수 있음, type에 text 주면 그냥 글씨 넣을 수 있고 	password로 하면 비밀번호 치는거처럼 ****로
button으로 버튼 만들 수 있음


html로 골격 만들었으면 css로 꾸며주기




<5. CSS 시작하기>
	<p id = "testid"> 했으면 <head>의 <style>에서 #testid{}로 해서 조작.
	id는 #으로 class는 .으로
	id는 홈페이지에 하나만(네이버 로고 같은거)
	class는 여러개(메일, 카페 이런거)

	border - 선, 1px 선의 넓이, solid 직선으로 하겠다.
	padding - 네모 안에서 민다
	margin - 바깥쪽에서 민다, margin-left는 왼쪽에서만 밀겠다


<6. CSS 좀 더 알아보기>
	<a> 에 target 옵션 줘서 단순히 링크 이동이 아닌 새창 띄우고 거기서 링크 이동하는 거처럼 할 수 있음


	<table>은 표 만든다고 생각하면 될듯
	<th>는 테이플 각 열의 이름 만드는거라 생각
	<tr>은 가로 한줄 행 <td>는 한 줄안에 들어가는 셀들

	이 스타일 빼서 새로운 파일로.
	<head>태그 사이에 <link> 만들어서 연결시킴


<7. 반응형 웹과 부트스트랩>
	<div>의 <style>에 width 줘서 창의 몇 퍼센트만 차지하게(창의 크기가 바뀔때 맞추서 계속 바뀌게)
	float 주면 세로로 붙이는게 아니라 가로로 붙일 수 있음

	부트스트랩 - css 하나하나씩 하면 너무 오래걸림, 미리 만들어둔 거 모아둠
